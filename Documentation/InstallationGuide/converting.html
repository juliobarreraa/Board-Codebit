Converting to IP.Board


To convert to IP.Board, then please do the following:

  • Install IP.Board as described in the Installation Guide
  • Download the latest converters from http://external.ipslink.com/ipboard30/landing/?p=converthelp
  • Upload all the files contained in the "upload" folder of the converters distribution
  • Go to your Admin CP, select the System tab, then choose Manage Applications & Modules
  • Under Applications Not Installed press the Install link for IPS converters
  • When the installation is complete, select My Apps and then IPS Converters
  • Follow the on-screen instructions

If you need assistance with the converting, we have a conversion service available. To find out more, see this page.
