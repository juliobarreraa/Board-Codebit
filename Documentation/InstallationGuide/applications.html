Installing Applications


To install an application such as IP.Blog or IP.Gallery, then please do the following:

  • Upload all the files contained in the "upload" folder of the application distribution
  • Go to your Admin CP, select the System tab, then choose Manage Applications & Modules
  • Under Applications Not Installed press the Install link for the appropriate application
  • The installation is now complete!
