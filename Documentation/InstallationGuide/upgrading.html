Updating your IP.Board and Applications


If you're updating your IP.Board or an application such as IP.Blog or IP.Gallery, then please do the following:

  • Important note: If you are using a database driver OTHER than MySQL please be sure to download the files for your driver from your client area and upload those files along with the files in the next step. If you fail to do this, you will not receive the SQL updates you need
  • Upload all the files contained in the "upload" folder of the download distribution. If you are upgrading multiple applications such as IP.Board and IP.Gallery, you should upload the contents of the "upload" folder from each download distribution.
  • After all the files have been uploaded, run the upgrade system by accessing "/admin/upgrade/index.php" on your system (for example: www.domain.com/forum/admin/upgrade/index.php) this will determine which upgrade modules need to be run and will rebuild your templates for you. Note: Remember to add "admin" into the upgrade URL as this was changed in verson 3.0 of IP.Board to offer a centralized upgrade center.
  • Your upgrade is now complete!
