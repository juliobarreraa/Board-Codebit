Deconstruct the URL

An array of URL bits are passed ( array( 'showtopic' => 100, 'pid' => '123' ) ). This function should attempt to return formatted data:

array( 'data_app' => '', 'data_type' => '', 'data_primary_id' => 0, 'data_secondary_id' => 0 )
