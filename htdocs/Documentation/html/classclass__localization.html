Generate Human formatted date string Return a date or '–' if the date is undef. We use the rather nice gmdate function in PHP to synchronise our times with GMT. This gives us the following choices: If the user has specified a time offset, we use that. If they haven't set a time zone, we use the default board time offset (which should automagically be adjusted to match gmdate.
