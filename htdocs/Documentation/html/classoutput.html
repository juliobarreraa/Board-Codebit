Fetch a skin based on user's incoming data (user-agent, URL) or via other params

Incoming Gateway file (index.php / xml.php / rss.php, etc) filters out some skins, then:

  • User Agent
  • URL Remap
  • App Specific
  • Member specific
  • Default skin
