Detailed Description

Simple Example: Import 20 articles that have #ipb in the title and 'ip.board' in the content store the guids we imported to prevent duplicate imports

$import = new rss_import( 'http://feed.com/feed.rss', 'blog-5' ); $import->load(); $import->build( array( 'title' => '#ipb', 'content' => 'ip.board', 'limit' => 20 ) );

while( $row = $import->fetch() ) {

Clean up topic title. Parse BBCode in content and add link back to source

print $import->cleanTitle( $row['title'] ) . '
' . $this->parseContent( $row['content'], $row['link'] ); }

$import->finish(); Advanced options

Don't add a link back $import->setLinkBack(false);

Allow HTML to be imported $import->setParseHtml(true);

Set http_auth details $import->setAuthUser('matt'); $import->setAuthPass('mypass');
