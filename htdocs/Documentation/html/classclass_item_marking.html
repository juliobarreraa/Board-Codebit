markRead (   $data,
  $app = '' 
)

Mark as read

Forum example (forumID is mapped to :item_app_key_1). itemID is the topic ID

$itemMarking->markAsRead( array( 'forumID' => 34, 'itemID' => 99 ) ); $itemMarking->markAsRead( array( 'forumID' => 34 ) ); If you pass in a timestamp, it will set that as the read date so a topic, for example won't appear read until you're on the last page $itemMarking->markAsRead( array( 'forumID' => 34, 'itemID' => 99, 'markDate' => 123456790 ) );

If you pass a "containerLastActivityDate" timestamp it will compare the markDate against it and not run the fetchUnreadCount query $itemMarking->markAsRead( array( 'forumID' => 34, 'itemID' => 99, 'markDate' => 123456790, 'containerLastActivityDate' => 1234567890 ) ); public
