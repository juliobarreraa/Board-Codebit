Detailed Description

MyMin - JSMin like alternative parser for JavaScript

This class is a jsmin alternative, based on same parser logic but modified to mantain performances and to parse correctly JavaScript conditional comments too.

SERVER SIDE PHP 5 or greater is required. This code is compatible with every error_reporting level (E_ALL | E_STRICT) The best practice to use this code is caching results without run-time evaluation (your server should be stressed too much with big files)
