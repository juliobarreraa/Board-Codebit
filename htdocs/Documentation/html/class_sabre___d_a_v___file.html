getETag ( )

Returns the ETag for a file

An ETag is a unique identifier representing the current version of the file. If the file changes, the ETag MUST change. The ETag is an arbritrary string, but MUST be surrounded by double-quotes.

Return null if the ETag can not effectively be determined
