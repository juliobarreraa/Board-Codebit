Detailed Description

Task to update the topic views from the temporary table.

Task to update the profile views from the temporary table.

Task to update data from Twitter and Facebook (photos, statuses, etc)

Task to remove locked members from the ACP list who are already unlocked.

Task to remove validating members over configured time.

Task to send out mobile notifications.

Task to send out bulk emails (dynamically enabled)

Task to send out the weekly digest emails.

Task to import RSS feeds defined in the ACP.

Task to send out the daily digest emails.

Task to prune daily old topic subscriptions.

Task to expire announcements.

Task to rebuild the board statistics.

Task to handle incoming emails through POP3.

Task to optimize database tables daily.

Task to cleanup old minify files.

Task to prune old logs from the cache folder.

Task to prune dead item markers.

Task to prune the content cache (posts & signatures)

Task to clean out 'dead' sessions, validations, registration image entires, etc.
